# Marketing deliverable kit — AXIOM Insurance Manager

Hand this folder to another AI agent or designer to build a deck, PDF, or one-pager.

## Quick start

1. Read `ASSEMBLY_INSTRUCTIONS.md`
2. Load slide copy from `MANIFEST.json` (structured) or `CONTENT.md` (prose)
3. Use `screenshots/*.png` where present; follow `screenshots/*.SPEC.md` to re-capture missing frames
